# Deckel FP4A post processor

Post processor to translate FreeCAD generated G-Code to Deckel FP4A with
Contour 2 control.

## Installation

Download the `deckel_post.py` file and move it to your FreeCAD Macro directory
(`C:\Users\<your_user_nmae>\AppData\Roaming\FreeCAD\Macro` on Windows,
`~/.local/share/FreeCAD/Macro` on Linux).
